# Day 26 - Model Comparison Framework

This project is part of my **100 Days of Machine Learning Challenge**.

On **Day 26**, I built a **Model Comparison Framework** to train and evaluate multiple machine learning models on the same dataset and compare their performance.

## 📊 Dataset
Titanic Dataset

## ⚙️ Features Used
- Pclass
- Sex
- Age
- Fare
- FamilySize
- Embarked

Basic preprocessing and feature engineering were applied.

## 🤖 Models Compared
- Logistic Regression
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Gradient Boosting

## 📈 Evaluation Metric
- Accuracy

## 🧠 Key Learning
Different algorithms perform differently on the same dataset.  
Using a **model comparison framework** helps identify the best-performing model for a problem.
