# Day 19 – XGBoost | #100DaysOfML

## Overview
On **Day 19** of my **#100DaysOfML** challenge, I learned about **XGBoost (Extreme Gradient Boosting)** — one of the most powerful and widely used machine learning algorithms for structured/tabular data.

## What I Learned
- Concept of **Boosting**
- Basics of **Gradient Boosting**
- How **XGBoost improves performance and speed**
- Handling **overfitting using regularization**
- Training and evaluating an **XGBoost model**

## Why XGBoost?
XGBoost is popular because it provides:
- High performance
- Built-in regularization
- Parallel processing
- Efficient handling of missing values

## Tools Used
- Python
- Pandas
- Scikit-learn
- XGBoost

## Key Takeaway
XGBoost is a powerful ensemble learning algorithm that combines multiple weak learners (decision trees) to create a strong predictive model.
